# 萨卡拉 Saqqara

> 人类最早的金字塔所在地——左塞尔阶梯金字塔（约 BC 2650），比胡夫大金字塔早 70 年，是整个金字塔建筑传统的起点。

## 概况

萨卡拉是位于开罗以南约 **30 公里**的大型古代墓葬遗址，历经古王国到托勒密王朝约 3000 年持续使用，是埃及保存最完整的王室墓地之一，2023 年依然有重大考古发现不断涌现。

**与吉萨同一天可游览**：从吉萨驱车约 40 分钟可到，适合半日游串联。

## 左塞尔阶梯金字塔

### 历史意义

| 项目 | 内容 |
|---|---|
| 法老 | **左塞尔（Djoser）**，第三王朝，约 BC 2650 年 |
| 建筑师 | **印和阗（Imhotep）**——古埃及最著名的建筑师兼医生，后被神化，希腊人将其等同于医神阿斯克勒庇俄斯 |
| 高度 | 约 62 米，6 层台阶结构 |
| 意义 | **人类历史上第一座大型石质建筑**，从平顶马斯塔巴（贵族墓）演化而来 |

### 建造逻辑

印和阗的天才之处在于：把六座逐层缩小的马斯塔巴叠放，创造出台阶形状——这既是技术实验，也是宗教象征（台阶是法老灵魂升天的阶梯）。之后达赫舒尔的弯曲金字塔、红色金字塔将台阶逐步平滑，最终到胡夫大金字塔达到完美平滑面。

```
马斯塔巴（平顶墓）→ 阶梯金字塔（叠加）→ 弯曲金字塔（尝试平滑）→ 红色金字塔（成功平滑）→ 胡夫大金字塔（巅峰）
```

## 萨卡拉的其他看点

| 遗址 | 内容 |
|---|---|
| **塞拉比姆（Serapeum）** | 神牛阿匹斯的巨型地下墓穴，长廊两侧排列花岗岩石棺，每具重 70 吨 |
| **提（Ti）墓** | 古王国官员墓，保存极佳的彩色浮雕，展现日常生活场景（捕鱼、农耕、造船） |
| **美热如卡（Mereruka）墓** | 33 间房的大型贵族墓，壁画内容极为丰富 |
| **乌纳斯金字塔** | 内壁刻有**最早的《金字塔文》**（约 BC 2350），是人类最古老的宗教文本之一 |
| **近年发掘** | 2022-2023 年持续发现大批木乃伊、彩棺、青铜器，部分文物已转移至大埃及博物馆 |

## 旅行建议

- 与吉萨、达赫舒尔可组成**"金字塔一日游"**串联行程
- 萨卡拉游客相对较少，比吉萨安静很多
- 塞拉比姆地下墓穴需要手电筒，光线很暗但极有震撼感
- 早晨光线最佳，阶梯金字塔在侧光下层次分明
